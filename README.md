# T1VibeCode – запуск проекта

## Требования
- Node.js 18+ и npm
- Python 3.11+ (рекомендуется использовать venv)

## Подготовка окружения
1. Скопируйте переменные окружения:
   ```bash
   cp .env.example .env.local
   # при необходимости заполните ключи LLM и другие переменные
   ```
2. Установите зависимости фронтенда:
   ```bash
   npm install
   ```
3. Создайте и активируйте виртуальное окружение для бэкенда:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```

## Запуск в режиме разработки
Откройте два терминала.

### Бэкенд (FastAPI)
Из корня репозитория:
```bash
python -m uvicorn main:app --reload --port 8000
```
При старте автоматически создастся/обновится SQLite-база и все необходимые таблицы.

### Фронтенд (Next.js)
Из корня репозитория:
```bash
npm run dev
```
Приложение будет доступно на http://localhost:3000.

## Полезное
- Конфигурация бэкенда находится в `server_py/` и файле `main.py` в корне.
- При изменении .env.local перезапустите соответствующий процесс.
- Для очистки node_modules или переустановки зависимостей используйте `npm ci` (если есть lock-файл).
